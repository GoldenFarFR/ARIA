# Mini-phrases prêtes à copier-coller pour Telegram (@Aria_ZHC_Bot)

Menu "/" : `/stop` et `/resume` (kill-switch). Les autres commandes admin
(`/status`, `/vc`, `/feuvert`, `/sentiment`, `/langue`, `/these`, `/issue`,
`/theses`, `/github`) restent fonctionnelles si tapées, juste retirées du menu.

Tout le reste = texte libre. Copie-colle une de ces phrases.

## État & suivi rapide
/status
Quel est ton état actuel ? (LLM, autonomie)

## Analyse & pronostics
Analyse ce token/wallet : [colle l'adresse]
/vc [adresse]
/feuvert

## Travail / délégation
Propose 3 idées concrètes pour améliorer Vanguard.
Fais un résumé des dernières conversations importantes.

## Mémoire & apprentissage (sans /learn)
Retiens ça : [ta leçon ou fait important]
La prochaine fois que je te demande X, fais Y.

## Discussion / idées
Donne-moi ton avis brut sur [sujet].
Comment on peut rendre Vanguard plus autonome ?
Quels sont les vrais blocages actuels selon toi ?

## GitHub (lecture seulement)
Liste les derniers changements dans le repo.
Y a-t-il des expériences intéressantes en cours ?

## Petit talk → ramener au projet
[Parle normalement] → elle ramène souvent vers Vanguard / autonomie.

Conseil : colle le contexte important directement dans le message (adresses, liens, chiffres).
