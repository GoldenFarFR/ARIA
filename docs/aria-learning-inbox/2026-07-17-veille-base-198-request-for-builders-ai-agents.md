## Fait nouveau : campagne nommée, priorités explicitement alignées ARIA

Le Base Ecosystem Fund a lancé une campagne nommée **"Request for Builders: Funding
the Future of Global Finance"** -- capital d'amorçage + ressources stratégiques pour
des équipes en pré-seed/seed. Secteurs prioritaires cités explicitement :
tokenisation RWA, finance stablecoin marchés émergents, crédit onchain, marchés de
prédiction, **et applications commerciales pour agents IA**. Source :
[crypto-economy.com](https://crypto-economy.com/base-launches-funding-initiative-to-accelerate-onchain-finance/).

**Pourquoi ça compte pour ARIA** : c'est la première fois qu'une campagne de
financement Base cite explicitement "AI Agents" comme secteur prioritaire nommé,
plutôt qu'une mention générique dans le pilier stratégique 2026 (déjà connu depuis
le 15/07). Renforce directement l'axe déjà retenu dans

(test 1M$ concluant) n'est pas atteint, mais confirme que la porte visée est la
bonne, pas une supposition.
