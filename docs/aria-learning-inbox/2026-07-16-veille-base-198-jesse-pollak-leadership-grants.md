## Fait nouveau : changement de leadership Base App (15-16/07)

Jesse Pollak a annoncé le 15 juillet qu'il se retire de la direction de l'équipe
**Base App** pour se consacrer exclusivement à l'infrastructure blockchain Base
elle-même (upgrades Azul, Beryl, B20, confidentialité, ledger). **Jordan Fish
("Cobie")** -- dont Coinbase a racheté le launchpad ICO Echo pour ~375M$ (cash +
actions) l'an dernier -- reprend la direction de l'app. Sources :
[The Block](https://www.theblock.co/post/408504/coinbases-jesse-pollak-hands-base-app-leadership-cobie-social-bets-fell-short),
[CoinDesk](https://www.coindesk.com/business/2026/07/15/coinbase-s-jesse-pollak-steps-back-from-base-app-leadership-after-admitting-his-crypto-social-strategy-failed).

Le post officiel Base confirme la doctrine des trois piliers en citation directe de
Pollak : *"Crypto is native money for computers, and artificial intelligence is going
to create trillions of new economic actors."* Source primaire :
[blog.base.org -- "The Agentic Economy Is Here"](https://blog.base.org/the-agentic-economy-is-here).

**Curiosité annexe, hors mission ARIA** : Pollak a aussi annoncé le lancement d'un
token personnel **$JESSE** sur Base App, avec un avertissement explicite de sa part
sur les risques d'usurpation/phishing autour de ce lancement (source :
[Cryptopolitan](https://www.cryptopolitan.com/jesse-pollak-to-launch-jesse-on-the-base-app/)).
Noté pour mémoire, sans lien direct avec la stratégie ARIA.
