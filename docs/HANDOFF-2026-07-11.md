générations de contrats coexistent) : le seuil n'est plus une constante globale mais
**par token** (`tokenGradThreshold(address)`). Formule validée empiriquement sur un vrai
token gradué : `reserve0 = 163 037 405 <= tokenGradThreshold = 168 316 832` — exactement
la condition de graduation du code source.
