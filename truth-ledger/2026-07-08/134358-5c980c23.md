## Question
et je veut aussi quelle demande dans quelle langue envoyer le rapport par mail

## Answer
Verdict : pas d'envoi de rapport par mail possible depuis ce canal — aucun skill email n'est branché sur le bot Telegram. Je ne vais pas te promettre un mail qui ne partira jamais.

Ce qui manque concrètement :
- Un skill mail (SMTP ou API type Resend/Postmark) configuré côté opérateur, avec la clé dans le coffre local.
- Une adresse destinataire.
- La source du rapport : là encore, watchlist Aria Market vide + pas d'on-chain live = pas de contenu à mettre dans le mail.

Sur la langue : bonne logique de vouloir la demander avant l'envoi. Une fois le skill mail en place, je peux poser la question à chaque rapport (FR / EN par défaut, puisque public = anglais, opérateur = français). Mais c'est prématuré tant que la brique mail et la source de données n'existent pas.

Ordre de priorité si tu veux vraiment ce pipeline :
1. Remplir la watchlist ou me donner la data brute.
2. Câbler le skill mail côté opérateur.
3. Je gère le choix de langue à l'envoi.

Tu veux que je scope la brique mail comme tâche worker, ou on règle d'abord la source de données ?
