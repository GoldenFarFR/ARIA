# aria-core

Package Python pour ARIA — runtime et sandbox.

## Installation

1. Cloner le dépôt ARIA (`GoldenFarFR/ARIA`) si ce n'est pas déjà fait.
2. Exécuter `.\install.ps1` dans le dossier `skills` pour installer les dépendances.
3. Configurer les variables d'environnement pour ARIA (`ARIA_RUNTIME`, `ARIA_SANDBOX`).
4. Lancer `.\run_aria.ps1` pour démarrer ARIA.

## Dépendances

- Python 3.11+
- Node.js LTS
- Git

## Sandbox

Le sandbox est utilisé pour les expérimentations et les tests. Il est configuré via le fichier `sandbox.json`.
- [repo-ingest] [ingest-repo] skills/aria-core/sandbox.json
{
  "runtime": "python",
  "env": {
    "ARIA_RUNTIME": "C:\\GitHub-Repos\\ARIA\\skills\\aria-core",
    "ARIA_SANDBOX": "C:\\GitHub-Repos\\ARIA\\skills\\aria-core\\sandbox"
  },
  "dependencies": [
    "python",
    "nodejs"
  ]
}
