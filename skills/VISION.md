## Role of this repo

| Aspect | Intent |
|--------|--------|
| **What** | Versioned Grok/Cursor skills — workflows, checklists, domain playbooks |
| **Why** | Distribution moat: same skills across vanguard, sandbox, future products |
| **Not** | Application code, secrets, or one-off prompts lost in chat |

## Principles

1. **One skill = one job** — focused `SKILL.md`, strong `description` for auto-invocation.
2. **Composable** — skills may reference ecosystem repos but stay installable standalone.
3. **Founder mode** — skills encode vision-aligned decisions (10x, moat), not generic boilerplate.
4. **English in skill bodies** when skills drive public/product output; French OK for internal ops skills.
